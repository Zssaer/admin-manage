## ⚡简介
 Admin通用后台。
 它是一个免费开源后台页面模板,使用Vue3、Vite、Element-Plus等主流技术和模块开发,可以达到下载少量编码即用，也是学习参考的好例子。
<hr>

## 👋安装使用
获取项目代码
```bash
git clone https://github.com/Zssaer/admin-manage.git
```
安装依赖
```bash
yarn install
```
运行
```bash
vite
```
打包
```bash
vite build
```
